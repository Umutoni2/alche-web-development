# HTML Advanced

---

## Project Overview
This project focuses on building a **complete webpage** using **semantic HTML elements only**.  
It follows a Figma design and serves as the foundation for future CSS styling and responsive design.

---

## Folder Structure
```

alche-web-development/
└── html_advanced/
├── README.md
├── index.html
└── layout-preview.png

```

---

## Steps

### 0. README and Objectives
- Create a `README.md` describing the project, structure, and objectives.

### 1. Header
- `<header>` with clickable logo (`<a><img></a>`) and 3 navigation links.

### 2. Banner
- `<main>` → `<section>` for banner.
- Headings, text, and 4 feature blocks (`img`, `h3`, `p`).

### 3. Quote
- `<section>` with `img`, `<blockquote>`, author `<p>`, subtext `<p>`.

### 4. Videos
- Section with `<h1>` + 4 video blocks:
  - `img`, `h2`, `p`, author info, rating stars.

### 5. Membership
- Section with `<h1>` + 4 membership items (`img`, `h2`, `p`) + button.

### 6. FAQ
- Section with `<h1>` + 2 rows × 2 items (`h2` + `p`).

### 7. Footer
- `<footer>` with logo, social links, and bottom text.
